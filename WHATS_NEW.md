# What's New / ما الجديد

## Version 3.0.0+5

### English (EN)
- **New Features:**
  - **About Page:** Added a dedicated page with developer contact info, GitHub links, and a "Donate via WhatsApp" button.
  - **Fix Data Errors Tool:** A new utility to check and repair database inconsistencies, specifically for customer balances. Accessed via the drawer.
  - **Debtors Card:** Added a new card on the Home Dashboard to quickly view outstanding debts.
  - **Extended Language Support:** The app now supports **16 languages** including French, Spanish, German, Italian, Portuguese, Russian, Hindi, and more!
- **Improvements:**
  - **UI Enhancements:** Fixed Android 15 Edge-to-Edge display issues.
  - **Translations:** Added missing translation keys (e.g., "Recent Transactions", "Fix Database Checks") to all supported languages.
  - **Bug Fixes:** Resolved crashes related to invoice deletion and custom item profit calculations.

---

### Arabic (AR)
- **الميزات الجديدة:**
  - **صفحة حول التطبيق:** تمت إضافة صفحة مخصصة تحتوي على معلومات المطور، روابط GitHub، وزر "تبرع عبر واتساب".
  - **أداة إصلاح أخطاء البيانات:** أداة جديدة لفحص وإصلاح تناقضات قاعدة البيانات، وخاصة لأرصدة العملاء. يمكن الوصول إليها عبر القائمة الجانبية.
  - **بطاقة الديون:** تمت إضافة بطاقة جديدة في لوحة التحكم الرئيسية لعرض الديون المستحقة بسرعة.
  - **دعم لغات موسع:** يدعم التطبيق الآن **16 لغة** بما في ذلك الفرنسية، الإسبانية، الألمانية، الإيطالية، البرتغالية، الروسية، الهندية، والمزيد!
- **التحسينات:**
  - **تحسينات واجهة المستخدم:** إصلاح مشاكل العرض "Edge-to-Edge" في Android 15.
  - **الترجمات:** إضافة مفاتيح الترجمة المفقودة (مثل "المعاملات الأخيرة"، "فحص قاعدة البيانات") لجميع اللغات المدعومة.
  - **إصلاح الأخطاء:** حل المشاكل المتعلقة بحذف الفواتير وحساب أرباح العناصر المخصصة.
